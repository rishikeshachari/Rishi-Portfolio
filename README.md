# Personal Portfolio Website

## 📌 Overview

This is my personal portfolio website built from scratch as Task 01 for the **Future Interns program**. It showcases my journey as an Electronics and Communication Engineering student at JNTUA (graduating 2027) and my growing skills in full-stack development.

The website serves as a central hub for my professional identity — featuring my technical skills, live projects, education timeline, and contact information. It bridges my background in hardware (VLSI, Arduino, IoT) with my passion for software development.

## ✨ Features

- ✅ **Fully Responsive** — Works seamlessly on desktop, tablet, and mobile devices
- ✅ **Dark / Light Mode Toggle** — User preference saved to localStorage
- ✅ **Smooth Scroll Animations** — Elements fade in as you scroll (Intersection Observer API)
- ✅ **Interactive Contact Form** — Client-side validation with real-time feedback
- ✅ **Projects Showcase** — Cards with tech tags and GitHub links
- ✅ **Education Timeline** — Visual timeline of my academic journey
- ✅ **Skills Section** — Categorized technical skills with hover effects
- ✅ **Optimized Performance** — Smooth transitions with `will-change` and `requestAnimationFrame`
