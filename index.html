Sky Shield — Autonomous Rain Protection System
✓ Fully functional hardware prototype
Designed and built a complete sensor-actuator system: a rain detector that automatically closes a roof and reopens it when conditions clear — no manual input needed.

Components: Arduino UNO, rain sensor module, 2 servo motors, limit switches for precision stop, LED status indicators. Wrote the full embedded C control loop — from sensor reading to motor actuation to feedback. This is my strongest technical project and demonstrates real hardware engineering thinking. 
// Sky Shield — sensor-actuator control loop
void loop() {
  int rain = digitalRead(rainSensorPin);

  if (rain == LOW) {      // Rain detected
    Serial.println("Rain: closing roof");
    closeRoof();           // Drive servos to closed pos
  } else {               // Clear — open roof
    openRoof();
  }
  delay(1000);
}
